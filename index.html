Synchronous.. Asynchronous.. ?

Latency impact can be reduced with asynchronous processing

Expressing latency with a type

scala.concurrent.Future[T]

Get synchronously

				
trait HttpClient {						
  def execute(request: Request): Response
}

// code will be blocked until response is returned
val response: Response = client.execute(someRequest)

Get asynchronously


trait HttpClient {						
  def execute(request: Request): Future[Response]
}

// Future[Response] is returned immediately
val f: Future[Response] = client.execute(someRequest)


// register callback to get the response when its ready
f onComplete { 
  case Success(response) => doSomethingInteresting(response)
  case Failure(throwable) => doSomething(throwable)
}
